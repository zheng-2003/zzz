一、MongoDB

MongoDB 是一个基于分布式文件存储的数据库。由 C++ 语言编写。旨在为 WEB 应用提供可扩展的高性能数据存储解决方案。

MongoDB 是一个介于关系数据库和非关系数据库之间的产品，是非关系数据库当中功能最丰富，最像关系数据库的。

mysql mongodb
数据库 数据库
数据表 集合
记录 文档

1、使用数据库

  • 查看有哪些数据库
  • 切换数据库
  • 删除数据库

2、使用集合

集合类似于我们mysql数据库的 表，用来存储记录的。

  • 查看数据库下所有的集合
  • 删除某集合

3、文档

文档类似我们mysql数据库的 记录，真正存储数据的地方

文档存储到 集合中。

3.1 插入数据

  • 插入一个文档到集合中

插入一条数据，返回插入的情况， acknowledged：代表已确认；insertedId 代表 插入Id。

  • 插入多个文档到集合中

插入了3条数据，则返回了3个对应的ObjectId

3.2 删除数据

remove() 函数是用来移除集合中的数据。

  • query :（可选）删除的文档的条件。
  • justOne : （可选）如果设为 true 或 1，则只删除一个文档，如果不设置该参数，或使用默认值 false，则删除所有匹配条件的文档。

3.3 修改数据

$set:关键字 用来修改

3.4 查询数据

查询语句find(条件)

3.5 条件操作符

因为我们的mongo 条件都是 {}字典，没有办法直接描述出 大于、小于等关系，因此需要 特殊的操作符来表示这些关系

  • (>) 大于 - $gt
  • (<) 小于 - $lt
  • (>=) 大于等于 - $gte
  • (<= ) 小于等于 - $lte
  • 案例1 查询 年龄大于30的学生
  • 案例2 查询 年龄 大于30 小于 40的学生

3.6 $type 操作符

mongodb中，同一个字段可能数据类型是不一样的，这样子，我们可以通过数据类型 来筛选

  • 筛选出所有名字是 数字的文档

3.7 limit使用

可以使用MongoDB的Limit方法，limit()方法接受一个数字参数，该参数指定从MongoDB中读取的记录条数。

  • 例如 取查询结果的前两条记录

3.8 skip使用

跳过几条记录，开始获取

  • 例如 从查询结果中 跳过2条获取结果

limit与skip组合在一起可以完成 分页的功能

  • 例如 一页2条记录，获取第二页数据
